You get the following mp3 and pdf files, if you download the fonetones Zip-File:

1) Ringtones of all the modern musical scales. The idea is that each scale can be used as a ringtone and each time the telephone rings the musical scale is heard and memorized. Example: A Major

2) Chords in all the scales, which can be used as envisaged at 1).
If the chords are memorized each, with the feeling it conjures, beautiful music can be written.
Chords were copied from Capone, Phil. 2007. 501 Guitar Chords. Cape Town: Struik. Example: F Minor 6 Chord.

3) PDF document for each fonetone's music notation. Example: A Major.pdf.
